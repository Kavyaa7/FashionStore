# FashionStore - Java Full Stack eCommerce Application

FashionStore is a full-stack eCommerce web application developed using Java technologies following the MVC architecture. The project allows users to browse products, view product details, add products to cart, and manage shopping flow dynamically.

## 🚀 Technologies Used

- Core Java
- JDBC
- JSP
- Servlets
- MySQL
- HTML
- CSS
- JavaScript
- Maven
- Apache Tomcat

## 🏗 Architecture

The project follows MVC Architecture:

- Model
- View
- Controller
- DAO Layer
- Service Layer

## ✨ Features

- User Registration & Login
- Dynamic Homepage
- Product Listing
- Product Details Page
- Add to Cart
- Cart Management
- Search & Filtering
- Sorting Functionality
- Responsive UI
- Session Management


## 🛒 Current Workflow

```text
Homepage
   ↓
Product Details
   ↓
Add To Cart
   ↓
Cart Page
```


## ⚙️ Setup Instructions

1. Clone the repository
2. Import project into Eclipse
3. Configure MySQL database
4. Update DB credentials
5. Run using Apache Tomcat Server
